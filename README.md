# Regression Linéaire en C++ From Scratch

## 1. Fonctionnalités

-Lecture d'un fichier csv sans entête 
- Calcul des gradients
- Mise à jour des coefficients a et b

## 2. Structure du dépôt:
RegressionLineaire.cpp : implementation des fonctions de de regression linéaire
RegressionLineaire.h: Déclaration des fonctions
main.cpp: Programme Principal, lecture du fichier csv, entraînement et prédiction
fic.csv: Données test, Dataset sans entête

## 3. Objectif:

La meilleure façon de s'entrainer et être sur de maitriser les bibliothèque classique c'est de pouvoir les 
reimplementer. Ce travail fait partie d'une série de travaux que je ferai ou je tente de reimplementer certaines 
bibliothèque souvent utiliser par les data scientist. 

Pour celui-ci, j'ai voulu faire en c++ qui est un langage que j'aime avec lequel je suis à l'aise. J'ai préféré travaillé avec les 
tableau de pointeur plutot qu'avec les vector pour aussi pratiquer un peu plus dessus.
